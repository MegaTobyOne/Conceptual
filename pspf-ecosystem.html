Workflow

The useful path is deliberately short: capture, validate, share, and recover. Every step keeps the authoritative policy source separate from local assessment data.

Step 1

Capture locally

Use Core and Workshop to maintain assessment state, evidence, actions, risks, Directions, ISM mappings, and planning context.

Step 2

Validate and export

Run validation, create snapshots, and export a master JSON bundle. The same bundle can support backup import through the validated review path.

Step 3

Review in Explorer

Open the bundle in Explorer for posture, records, Connected View, saved views, local changes, and a copyable posture brief.
