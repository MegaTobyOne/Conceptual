# PSPF Pub

PSPF Pub is the local-first people, role, team, assignment, and stakeholder relationship surface for PSPF workspaces.

The v1.29 release keeps Pub publishable as an independent VS Code extension and completes local detail/edit coverage for Person, Role, Assignment, and Relationship Note records. Pub data is not exported into Explorer bundles; sensitive person details, relationship notes, development context, performance-management context, roster planning, rotations, anniversaries, and team-event history stay local-only by default.

Pub depends on PSPF Core and uses the `pspf.pub.*` command namespace. The package provides the Activity Bar entry, Home view, Organisation Chart, Teams, People, Roles, Assignments, and Relationship Log views, plus local-only detail/edit panels for the Pub record types.

## Key commands

- `PSPF: Open Pub`
- `PSPF: Load Pub Sample`
- `PSPF: New Pub Person` / `PSPF: Open Pub Person Detail` / `PSPF: Edit Pub Person`
- `PSPF: New Pub Role` / `PSPF: Open Pub Role Detail` / `PSPF: Edit Pub Role`
- `PSPF: New Pub Assignment` / `PSPF: Open Pub Assignment Detail` / `PSPF: Edit Pub Assignment`
- `PSPF: Record Pub Relationship Note` / `PSPF: Open Pub Relationship Note Detail` / `PSPF: Edit Pub Relationship Note`
