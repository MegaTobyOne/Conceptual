# PSPF Pub

PSPF Pub is the local-first people, role, team, assignment, and stakeholder relationship surface for PSPF workspaces.

The v1.28 foundation release makes Pub publishable as an independent VS Code extension and establishes the product surface without exporting Pub data into Explorer bundles. Sensitive person details, relationship notes, development context, performance-management context, roster planning, rotations, anniversaries, and team-event history stay local-only by default.

Pub depends on PSPF Core and uses the `pspf.pub.*` command namespace. The first package shell provides the Activity Bar entry and foundation view for later organisation-chart, action badge, assignment, and relationship-management workflows.
